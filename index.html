简单的hexo教程

发表于 2018-10-23

前言

我为什么选择了hexo，选择创建一个属于自己的博客？
在某王同学的安利下，选择了hexo这个看起来很高大上的框架
一开始还很zz的想，作为一个合格（刚入门）的前端工程师，博客肯定要自己写啊
然后emm发现，我连hexo都玩不转= =
话说回来，学习和输出总是相得益彰的吧，有所学，有所写，更能有所获吧，希望能以此来鞭策自己，也给自己的生活添加一点趣味吧。

什么是hexo？

hexo是一款基于node.js（我暂时玩不转）的静态博客框架，可以很方便的生成静态网页托管在GitHub上。
下面正式开始搭建属于我们自己的博客。

搭建步骤

  • 注册GitHub,创建个人仓库
  • 安装git，并且在廖雪峰和githug的调教谆谆教诲下存活下来
  • 安装Node.js
  • 安装hexo
  • 渲染网页
  • 更换主题
  • （学习markdown语法）开始推文
  • 寻找图床
  • 个性化设置

注册GitHub

welcome to全球最大的开源代码库
如果你还没有自己的账号的话，click here，使用自己的邮箱注册一个
戳用户名下的Your repositories，创建一个新的仓库
如果这个仓库要用于托管我们的hexo的话，仓库名为：用户名.github.io 用户名为你的GitHub账号名称。
例如，我的GitHub名为MaxJ2000，我的仓库名为：MaxJ2000.github.io
这个仓库名，就将成为你博客的访问网址

安装git

什么是Git?简单来说Git是开源的分布式版本控制系统，用于敏捷高效地处理项目。
在百度下载好Git后，开始学习如何使用这个强大的工具吧！传送门
如果你觉得自己还要多加练习的话，这里有一个有趣的通关小游戏GitHug
通关后，你已经熟悉了Git的所有常用操作了！

安装node.js

日常百度下载后，win+r 打开cmd，输入
node -v
检测node安装是否成功
成功后，在cmd中输入npm -v
如果出现版本号的话，证明安装成功了。
现在开始我们的正事，hexo的安装。
在任意地方创建一个文件夹，命名为MyBlog，我们即将在这里存放你的博客框架以及内容。
进入文件夹中，按住shift，右键，打开命令行
输入
npm install -g hexo-cli
安装完成后，初始化我们的Blog，输入：
hexo init blog
